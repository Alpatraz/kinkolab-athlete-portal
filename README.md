# KinkoLab - Portail Athlètes

## Installation locale

```bash
npm install
npm run dev
```

## Build Netlify

```bash
npm run build
```

Déposer ensuite le dossier `dist/` sur Netlify.

## Important

Remplacer `src/App.jsx` par le code complet du canvas :
`Portail Athlete Kinko Lab App Repare`.

## Paramètres Netlify si déploiement GitHub

- Build command : `npm run build`
- Publish directory : `dist`
- Node version : `20`
